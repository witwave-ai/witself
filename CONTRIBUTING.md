# Contributing To Witself

Status: pre-implementation draft.

Witself is currently in a maintainer-directed design phase. Public visibility
does not yet mean the project is accepting arbitrary code contributions or
external security-architecture changes.

Before contributing, read:

- [docs/requirements.md](docs/requirements.md)
- [docs/access-policy.md](docs/access-policy.md)
- [docs/governance-and-support.md](docs/governance-and-support.md)

Security issues must be reported privately through [SECURITY.md](SECURITY.md),
not through public issues or pull requests.

Implementation contribution rules, local development commands, test commands,
lint commands, and CLA or DCO policy will be added before the first public code
contribution path is opened.
